# 制作背景
- 美味しい焼肉を食べたい！  
焼肉店の多くは私達が最終調理をします。特に和牛は、焼き加減一つで味わいが変わってきます。  
部位によっても焼き加減が難しく、業界に関わっていた私自身も最適な焼き方を把握していませんでした。  
そこで、部位を入力すれば最適な焼き方を提案してくれるBotを作成しました。
